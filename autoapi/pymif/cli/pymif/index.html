pymif.cli.pymif.zarr_convert(input_path, zarr_path, microscope: str | None = None, max_size: int | None = 100, chunk_size: List[int] | None = None, scene_index: int | None = 1, channel_names: List[str] | None = None, channel_colors: List[str] | None = None, zarr_format: int | None = 3, downscale_factor: int | None = 2, num_levels: int | None = None)[source]

Helper function for CLI to convert a dataset to zarr given some parameters.

Parameters

input_pathstr

Input path for the data to be converted.

zarr_pathstr

Output .zarr path.

microscopestr

Microscope used to acquire input data. One of “luxendo”, “opera”, “viventis”, “zeiss”, “zarrv04”, “zarr”.

max_sizeOptional[int]

Max chunk size in MB.

Default: 100

chunk_sizeOptional[List[int]]

Chunk size in TCZYX format, or whatever axes are present in the dataset. Default: None

scene_indexOptional[int]

Scene index for .czi files.

Default: -1

channel_namesOptional[List[str]]

Name of channels.

Example: “-cn bf gfp rfp”

Default: None

channel_colorsOptional[List[str]]

Colors of channels (hex or matplotlib color name)

Example: “-cc 0000FF cyan 00ff00”)

Default: None

zarr_formatOptional[int]

Output zarr format. Zarr v2 maps to NGFF 0.4 and Zarr v3 maps to NGFF 0.5.

Default: 3

downscale_factorOptional[int]

Pyramid downsampling factor.

Example: “-df 1 2 2”)

Default: 2

num_levelsOptional[int]

Number of pyramid levels.

Example: “-nl 3”)

Default: None
