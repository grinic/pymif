class pymif.microscope_manager.ZarrV04Manager(path, chunks: Tuple[int, Ellipsis] | None = None, metadata: dict[str, Any] | None = None)[source]

Bases: pymif.microscope_manager.zarr_manager.ZarrManager

Backward-compatible reader for legacy NGFF/OME-Zarr v0.4 stores.

The original implementation assumed five-dimensional TCZYX data. This compatibility wrapper now delegates to ZarrManager, which supports both v0.4 and v0.5 metadata layouts and any unique subset of the image axes t, c, z, y and x.

Create a MicroscopeManager instance from in-memory NumPy or Dask array(s) with user-defined metadata. Supports single resolution or multiscale pyramid.

The default remains legacy TCZYX for 5D arrays, but metadata['axes'] may be any unique combination of t, c, z, y and x whose length matches the input arrays.
