pymif.microscope_manager.utils.pyramid.build_pyramid(data_levels: List[dask.array.Array], metadata: Dict[str, Any], num_levels: int = 3, downscale_factor: SpatialFactor = 2, start_level: int = 0) → Tuple[List[dask.array.Array], Dict[str, Any]][source]

Generate a multiscale pyramid and updated metadata for NGFF-compatible OME-Zarr writing.

This function is axis-aware and uses metadata[“axes”] to find Z, Y, X.

downscale_factor can be either:

  • 2, meaning downsample Z, Y and X by 2

  • (1, 2, 2), meaning keep Z unchanged and downsample only YX
