Internal function that writes image data (pyramid or single array) to a zarr group.

If a single array is provided, the function automatically generates the complete pyramid by upsampling and downsampling from the given level.

Parameters

rootzarr.Group

The root Zarr group.

modestr

Zarr open mode (“r+”, “a”, or “w”). Must allow writing.

datanp.ndarray, da.Array, or list thereof

Array(s) to write. If a list, each entry corresponds to one resolution level. If a single array, pyramid levels will be generated automatically.

t, c, z, y, xint or slice

Indices/slices for each dimension.

levelint

Pyramid level of the provided data if data is a single array.

group_namestr, optional

Name of the subgroup inside the root. If None, writes to root image.
