.. py:function:: zarr_convert(input_path, zarr_path, microscope, max_size: Optional[int] = 100, scene_index: Optional[int] = -1, channel_names: Optional[List[str]] = None, channel_colors: Optional[List[str]] = None)

   Helper function for CLI to convert a dataset to zarr given some parameters.

   Parameters
   ----------
       input_path : str
           Input path for the data to be converted.
       zarr_path : str
           Output .zarr path.
       microscope : str
           Microscope used to acquire input data.
           One of "luxendo", "opera", "viventis", "zeiss", "zarrv04", "zarr".
       max_size : Optional[int]
           Max chunk size in MB. 

           Default: 100
       scene_index : Optional[int]
           Scene index for .czi files. 

           Default: -1
       channel_names : Optional[List[str]]
           Name of channels.

           Example: "-cn bf gfp rfp"

           Default: None
       channel_colors : Optional[List[str]]
           Colors of channels (hex or matplotlib color name)

           Example: "-cc 0000FF cyan 00ff00")

           Default: None
